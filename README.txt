X-Ray Tracker PRO V2 (Neon) — Installable Web App (PWA)

Files:
- index.html (the tracker)
- manifest.json (app metadata)
- sw.js (offline cache)
- icon-192.png, icon-512.png (app icons)

How to install as app:
1) Upload this folder to any HTTPS hosting (Netlify / GitHub Pages / Firebase Hosting).
2) Open the URL in Chrome on Android.
3) Tap 'Install app' or menu → 'Add to Home screen'.

APK route (no PC option is limited):
- Fastest: Use PWABuilder.com → enter your hosted URL → Build → Android.
- Or use WebIntoApp.com to wrap the hosted URL into an APK.

Note: Android Chrome only shows Install when the site is served over HTTPS (not file://).
